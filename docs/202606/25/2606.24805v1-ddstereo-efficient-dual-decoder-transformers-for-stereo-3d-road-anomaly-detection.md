## 摘要
基于立体的3D目标检测仍面临两个关键安全挑战：实时性能和开集泛化。现有的立体3D方法通常能达到单目方法两倍的精度，但推理速度显著降低，不适合实时应用。同时，近来开放世界检测的进展在单目2D和3D场景中引入了开集和开放词汇算法，但基于立体的开集检测仍然鲜有探索。为弥补这一差距，我们提出DDStereo，一种用于实时开集3D目标检测的新型双解码器立体Transformer。DDStereo具有两个轻量级解码器分支：一个用于开集前景2D检测，另一个用于3D属性回归。这些解码器共享目标级查询以实现统一的目标级对齐。为提高推理效率，我们设计了一个紧凑的视差特征提取器和流线型解码器架构。在公开的立体3D基准上的实验表明，DDStereo在闭集和开集协议下均达到最先进精度。值得注意的是，我们的方法在推理速度上超越现有立体3D检测器，并首次实现了与单目方法相媲美的实时性能。

## Abstract
Stereo-based 3D object detection still faces two critical safety challenges: real-time performance and open-set generalization. Existing stereo 3D methods typically achieve twice the accuracy of monocular methods but suffer from significantly lower inference speeds, making them unsuitable for real-time applications. Meanwhile, recent advances in open-world detection have introduced open-set and open-vocabulary algorithms in monocular 2D and 3D settings, yet stereo-based open-set detection remains largely unexplored. To bridge this gap, we propose DDStereo, a novel Dual-Decoder Stereo Transformer for real-time open-set 3D object detection. DDStereo features two lightweight decoder branches: one for open-set foreground 2D detection and the other for 3D attribute regression. These decoders share object-level queries to achieve unified target-level alignment. To enhance inference efficiency, we designed a compact disparity feature extractor and a streamlined decoder architecture. Experiments on public stereo 3D benchmarks demonstrate that DDStereo achieves state-of-the-art accuracy under both closed-set and open-set protocols. Notably, our method surpasses existing stereo 3D detectors in inference speed and, for the first time, achieves real-time performance comparable to monocular approaches.

- **背景**：单目3D检测已出现开集/开放词汇方法（如OV-Mono3D、DetAny3D），但立体3D领域的开集检测几乎是空白。已有方法S3AD虽提出基于立体的OoD检测，但推理速度超过80ms，不适合实时。此外，现有开放词汇方法通常需要文本提示、点或框等额外输入，而自动驾驶更急需的是只依赖视觉输入的二分类OoD检测（判断是否有障碍物）。
- **整体意义**：本文提出首个能同时实现**实时推理**和**开集检测**的立体3D检测器，填补了立体开集检测与实时性之间的空白，推动立体视觉在安全关键的自动驾驶中实用化。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：将开集3D检测解耦为三个子任务：常规的多类别检测、新类别的二分类检测、实时检测。为此设计**双解码器架构**：
  - **前景检测分支**：基于视差特征（disparity features），使用二进制DETR风格解码器预测所有潜在物体的2D框和前景概率（类别无关，受几何线索驱动）。
  - **3D检测主分支**：基于左图视觉特征，使用Transformer编码器-解码器预测3D尺寸、朝向、已知类别得分和2D框（使用投影中心参数化）。
  - **共享目标查询（Shared Queries）**：两个解码器共享同一组可学习目标查询，保障空间一致性和目标级对齐，避免了NMS或锚点匹配。
- **关键技术细节**：
  - **视差特征提取**：计算左右特征的相关体（Correlation Volume），水平位移最多d像素，在多个尺度上操作（d=24/16等），生成紧凑的视差特征图fD。
  - **深度图预测**：通过U型卷积网络从多尺度相关体重建高分辨率（1/4原图尺寸）深度图，然后在3D分支预测的物体中心使用网格采样（Grid Sample）获取物体深度，代替复杂的深度引导交叉注意力。
  - **开集置信度估计（MNPF）**：  
    \[ S_{\text{OoD}} = S_f - \max_{i \in K} \sigma(S_i) \]  
    其中 \(S_f\) 为前景置信度，\(\sigma\) 为sigmoid函数，\(\max \sigma(S_i)\) 为已知类别中的最大归一化概率。若差值大则判定为OoD。
  - **损失函数**：包含2D前景损失（二进制分类+回归）、3D检测损失（类别、朝向、尺寸、3D中心、2D框）、深度图与视差监督损失，各部分独立匈牙利匹配。
- **算法流程**：输入左右图像→共享Backbone提取多尺度特征→左图特征用于3D分支，左右相关体用于视差特征→两个解码器（共享查询）分别输出前景2D框+前景概率和3D属性+已知类别→MNPF计算OoD得分→后处理得到最终3D检测。
